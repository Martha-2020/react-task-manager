# React Task Manager

A responsive task management application built with React that helps users organize their daily tasks efficiently.

## Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed
- 🔍 Search tasks
- 📂 Filter by All, Completed, and Pending
- 🚦 Priority levels (High, Medium, Low)
- 📅 Set due dates
- ⚠️ Overdue task indicator
- 💾 Local Storage persistence
- 📱 Responsive design

## Technologies Used

- React
- JavaScript (ES6+)
- CSS3
- Local Storage
- Vite

## Installation

```bash
git clone https://github.com/Martha-2020/react-task-manager.git
cd react-task-manager
npm install
npm run dev
```
